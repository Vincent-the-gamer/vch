> [!WARNING]
> Under construction.

## Why vch?

I think I'm a little obsessive-compulsive about keeping my dev toolchains newest(or at latest LTS version), like node, uv, brew, git and etc. I'm always checking versions from the official sites of them.

So, why not to check them together?

## Usage

No need to install, run it with npx

```bash
npx vch
```
